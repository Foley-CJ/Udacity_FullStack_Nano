Fractal Project

This project explores the mathematical concept of fractals. It specifically focuses on the Sierpinski triangle. The code was written in python 3 and heavily relies upon the turtle package. It accepts a integer parameter that represents the depth of the triangle formation. Although there is no limit, the time it takes for the fractal generation increases exponetially and should be limited to a max depth of 9. The code itself loops recursively for the given depth splitting the triangles at their mid points

Movies Project

This is a basic movie website. There are three main python files involved, data, class, tomatoes. The data file holds the raw data associated with each movie. All of this data is stored into a list of objects. The class movie forms the data object that the data file leverages to store the data in an organized manner. The tomatoes file imports the data from the raw file and generates a single html file which presents the data

Mall Analytics

This is a placeholder for an ongoing project. The project relies heavily on python and spark to complete big data analytics tied to Mall usage. The project is split into three components with the majority spent on the latter two: Data collection, grooming, and visualization. The data visualization developement is being conducted with a variety of python packages including bokeh and plotly while final production will leverage tableau.

Memory Game

This ongoing project is a memory game built on python's django framework and the d3 javascript collection. Originally built as a test for data sessions it features a random question generator written in python that requires the gamer to recall the color, order, and type of shapes shown in previous pages. Future improvements to the game include difficulty settings and a multiplayer mode.
